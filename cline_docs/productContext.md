# Product Context

## 项目存在原因

AI Slots 是一个 AI 驱动的老虎机主题生成器游戏平台。核心价值在于让运营/用户用一句话描述即可自动生成完整的老虎机主题，大幅降低游戏内容的制作门槛。

## 解决的问题

传统老虎机主题开发需要美术、策划、开发多方协作，周期长、成本高。本项目通过 AI 自动生成主题文案和符号图片，将主题制作时间压缩到分钟级。

## 工作方式

1. 用户输入一句话描述（如"生成一个古埃及风格主题"）
2. DeepSeek 生成主题名称、背景故事、符号列表和美术风格词
3. 即梦 API 并发生成 11 张符号底图（干净无特效）
4. 主题配置写入 MongoDB，图片存储到本地/OSS
5. 前端提供完整老虎机游戏体验（PixiJS 渲染，WebSocket 实时推送）
6. 游戏内特效（发光、粒子、连线高亮）由 PixiJS 动态叠加
